🚀 HOW TO USE IT:

Step 1: On Your Windows Development Machine
────────────────────────────────────────────

Requirements:
  ✓ Python 3.11+ (download from python.org)
  ✓ Inno Setup 6.0+ (download from jrsoftware.org/isdl.php)
  ✓ 500 MB free disk space

Step 2: Build the Installer
───────────────────────────

Option A: Professional Installer (Recommended for distribution)
  1. Navigate to: app_mother/biblioteca-app/
  2. Double-click: build_installer.bat
  3. Wait 5-15 minutes for build completion
  4. Find: installer/output/RegistruDigital_Setup_1.7.0.exe
  5. Share this .exe file with your users!

Option B: Portable Executable (No installation needed)
  1. Double-click: build.bat
  2. Find: dist/RegistruDigital/RegistruDigital.exe
  3. Users extract and run directly
